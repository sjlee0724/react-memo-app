# 메모 앱

토이프로젝트로 만들어본 메모 앱입니다.

## 설치(Installation) 및 사용 방법(How to use)

---

### 설치 방법(How to install)

```bash
npm install
or
yarn install

npm start
or
yarn start
```

### 사용 방법(How to use)

새로운 메모를 작성하고 싶다면 new memo 버튼을 클릭하면 사용할 수 있습니다. <br>
수정하고 싶은 메모가 있다면 클릭 후 수정한 다음 Save 버튼을 누르면 됩니다.<br>
